| for | String | false | | 绑定控件的 id | true |

用来改进表单组件的可用性，有两种使用方式：
1. 使用 `for` 属性找到对应的 `id`
2. 将控件放在该标签下，当点击时，就会触发对应的控件

`for` 优先级高于内部控件，内部有多个控件的时候默认触发第一个控件

目前可以绑定的控件有：`XButton`、`XCheckbox`、`XRadio`、`XSwitch`
